Ces fichiers sources permettent de réaliser une utilisation de l`API "OpenStreetMap". Cette API <br>
permet de récupérer les informations concernant à la géolocalisation.<br>

Ce que j`ai fait :<br>
  - J`ai d`abord crée un bouton (avec des animations avec le curseur, etc...)
  - Lorsqu`on clique sur ce bouton, il affiche un tableau (stylisé par BootStrap).<br>
  Il indique les informations différentes (Pays, Ville, Région, Département, N° de la rue,<br>
  Nom de la rue, etc...). Ces informations sont récupérée à l`aide de l`API.
  - L`API communique aussi dans la console avec l`aide JSON (j`affiche les informations par un<br>
  tableau (fetch).
